# Jump Script

A bash function to move up the directory tree by folder name.

## Example

Current path  
/home/kareem/projects/backend/api/src

Command  
jump projects

Result  
pwd -> /home/kareem/projects

## Setup

Load the function into the current shell.

```
source jump.sh
````

## Usage

```bash
jump folder_name
Note: The folder must exist in the current path.
```
